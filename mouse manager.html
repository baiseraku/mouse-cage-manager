导出数据

↓ 文件保存位置说明

JSON 会导出全部笼位原始数据；CSV 和 PDF 会导出当前勾选的笼位。

浏览器出于安全限制，无法由网页指定下载路径。文件将保存到浏览器的默认下载文件夹。

如需更改保存位置，请在浏览器中设置：

Chrome：设置 → 下载内容 → 更改下载位置

Edge：设置 → 下载 → 更改下载文件夹

Firefox：设置 → 常规 → 下载 → 选择保存位置

也可在浏览器下载设置中勾选 "每次下载都询问保存位置"，这样每次导出时都会弹出文件夹选择框。
